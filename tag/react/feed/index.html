Once I discovered the power and flexibility of this JavaScript library, I knew that I would never be able to go back to working with vanilla JavaScript. React has completely transformed the way I approach web development, and has opened up new possibilities and opportunities for me as a developer. In this article, I’ll share my journey to learning React, including the challenges, breakthroughs, and insights that I gained along the way.

I started my journey to learning React by enrolling in Scrimba’s 10-hour React course. This course provided a solid foundation for understanding the basics of React, including components, props, state, and lifecycle methods. I also found SuperSimple Dev’s videos to be incredibly helpful. Their step-by-step tutorials and practical examples helped me apply what I learned in Scrimba’s course to real-world scenarios.

However, despite the quality of the resources I used, I still encountered some challenges along the way. One of the biggest hurdles I faced was getting the React package installed and configured properly on my machine. This resulted in a lot of frustration and wasted time, and forced me to do some of my early coding exercises on StackBlitz. Additionally, I found the JSX syntax, particularly the use of props, to be quite difficult to acclimate to. It took me some time to understand how to properly pass data between components, and how to structure my code in a way that was both efficient and readable.

Despite the challenges I faced, I continued to persevere in my journey to learn React. In fact, I’ve already made significant progress with a personal project I’m working on: a neural network builder app. Using the skills I’ve learned through Scrimba’s course and SuperSimple Dev’s tutorials, I’ve been able to create a dynamic app that allows users to create and delete nodes in a neural network, as well as turn them on and off. This has been an exciting and rewarding experience, and has really solidified my understanding of React’s core concepts.

Through my experiences learning React, I’ve also come to understand the importance of having the right tools for the job. My current laptop, while functional, is not ideal for running resource-intensive development environments, and has slowed me down at times. Additionally, I’ve realized that proper code organization and structure is crucial for building complex applications with React. Specifically, I need to gain a better understanding of JavaScript modules and how they can be used to modularize my code. Most importantly, I’ve learned that watching videos alone isn’t enough to master a new skill. Learning by practice and building actual projects is essential to cementing concepts and building confidence in my abilities.

In conclusion, my journey of learning React has been an enriching and rewarding experience. I have gained a solid understanding of the basics of React and have already begun developing my own neural network builder app using React. Although I encountered some difficulties along the way, such as struggling to install packages and acclimatizing to the JSX props syntax, I persevered and was able to overcome these obstacles. Going forward, I plan to continue developing my skills and building more complex applications with React. Of course, if I find a better library that suits my needs, I may switch to it, but for now, React is the perfect tool for me. I cannot stress enough the importance of learning by practice when it comes to mastering any new skill, and React is no exception. With practice, patience, and persistence, anyone can become a proficient React developer.
