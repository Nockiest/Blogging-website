After building several projects, I felt confident enough to start my own game development journey. In this article, I will share my experience of building a clicker defense game using React and GPT.

The Idea

My inspiration for this game came from playing simple flash games that I found enjoyable. I wanted to create a game with simple mechanics but challenging gameplay.

This is what my game will be about

  • Genre: A kind of tower deffense combined with a clicker game
  • Goal: Survive until wave 40
  • Game mechanics: Enemies will spawn at the top of the game field and advance closer to your base. By reducing their hitpoints they will be killed. This is achived through clicking on them, or placing towers to shoot for you
  • Features: Proggresively harder waves, Reloading you “clicks”, support towers, traps, boss, audio and more

Setting Up the Game Field and Settings Bar

I started by setting up the game field and the settings bar layout in vanilla JavaScript using a few line prompt. However, with GPT’s recommendation, I switched to using React. This allowed me to break down the project into smaller, more manageable components, making it easier to organize and maintain the codebase.

Creating Enemies

After setting up the game field, I created an enemy class and rendered it on the canvas. Then, I tried to kill them using my mouse. It took some fiddling with the calculation of the click coordinates, but I managed to kill them when I clicked on their sprite.

Movement and More

Next, I added movement to the enemies by setting them on top of the game field and making them drop down every 0.01 second by subtracting a speed value from their y index. With everything working, I decided to call it a day for now. However, I will continue to develop my game by adding new features and graphics.

!So that is the current state for now!

Conclusion

With the help of GPT, I was able to switch from vanilla JavaScript to React, which allowed me to build a more scalable and robust game. While there were some challenges along the way, the end result was worth it. I look forward to continuing my game development journey and sharing updates with you.
